# Portfolio Profissional — Carlos André

Portfolio profissional de Carlos André Almeida de Oliveira, desenvolvido com React, Vite e Tailwind CSS para apresentar projetos reais, habilidades técnicas, formação acadêmica e trajetória profissional na área de tecnologia.

## Visão Geral

Este projeto centraliza a presença profissional de Carlos André em uma interface moderna, responsiva e objetiva. O portfolio destaca experiências práticas em desenvolvimento full-stack, sistemas financeiros, plataformas administrativas e aplicações web em produção.

## Tecnologias Usadas

- React
- Vite
- Tailwind CSS
- JavaScript
- HTML5
- CSS3

## Funcionalidades Principais

- Apresentação profissional com seção hero.
- Listagem de projetos destacados.
- Seções de habilidades técnicas, formação e diferenciais.
- Área de contato com links sociais centralizados.
- Layout responsivo para desktop e dispositivos móveis.
- Build otimizado com Vite.

## Projetos Destacados

- **FinSync**: plataforma de inteligência financeira e gestão pessoal.
- **PayTech PDV**: sistema de ponto de venda e estrutura operacional.
- **Instituto Elo Libras**: sistema de inscrição e gerenciamento administrativo.

## Como Instalar Dependências

```bash
npm install
```

## Como Rodar Localmente

```bash
npm run dev
```

Depois acesse a URL local exibida pelo Vite no terminal.

## Como Gerar Build

```bash
npm run build
```

Para visualizar o build localmente:

```bash
npm run preview
```
